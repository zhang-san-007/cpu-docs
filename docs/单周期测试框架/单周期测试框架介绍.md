该项目是一个RISC-V单周期处理器仿真、测试框架。你可以将你的单周期处理器接入到该框架中，用于验证自己的处理器实现是否正确，也可以借助该框架使用你自己的处理器运行一些简单的软件程序。

支持测试的指令集：RV32IM

**项目目录如下**:
```
single-cycle-cpu
├── abstract-machine                 # 裸机运行时环境(目前忽略它)
├── IP                               # 处理器代码目录
│    ├── mycpu                          # 此目录用于接入你的单周期处理器
│    ├── single-cycle-cpu               # 项目自带的一个单周期处理器(verilog)
│    └── single-cycle-cpu-master        # 项目自带的一个单周期处理器(verilog)
├── software-test                    # 测试处理器的软件程序
│    ├── benchmarks                     # benchmark测试程序
│    │   cpu-tests                      # 简单cpu测试程序
├── simulator                        # 处理器模拟仿真框架
│    ├── include                        
│    │   nemu                           # 用于difftest的nemu动态链接库目录
│    │   src                            # 源代码目录
│    └── Makefile                       
├── Makefile                         
└── README.md

```
