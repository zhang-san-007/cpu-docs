
你需要自行搜寻资料来安装ubuntu 22.04 LTS, 安装Ubuntu 22.04 LTS有如下几种选择：

- windows&ubuntu双系统
- WSL(该项目支持WSL环境)
- 通过虚拟机使用ubuntu


!!! tip "记得更换Ubuntu软件源"
    由于 Ubuntu 默认软件源在国内访问速度较慢，我们建议大家更换为国内的软件源，例如[北外镜像源](https://mirrors.bfsu.edu.cn/help/ubuntu/)。
    
    更换软件源后，使用 sudo apt update 命令更新索引。
